# open* — A curated map of Open* projects worth contributing to

A personal learning and contribution tracker for open-source projects whose name starts with **open*** — OpenBMC, OpenInfra, OpenStack, OpenTelemetry, and many more.

Each entry is verified as **truly open source** (OSI-approved license, public repo, active community).

---

## How to use this

1. Browse the categories below
2. Click a project → read its `projects/<name>.md` file for contribution tips
3. Start with **Difficulty: Beginner** entries
4. Track your progress in [`MY_JOURNEY.md`](MY_JOURNEY.md)
